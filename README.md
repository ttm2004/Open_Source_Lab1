## 📝 Giới thiệu
Đây là repository chứa các bài tập thực hành PHP cơ bản trong môn học Open Source.

## ⚙️ Các chức năng đã thực hiện

### 🔹 Bài 1: Form nhập tên
- Nhập tên người dùng
- Hiển thị lời chào bằng JavaScript alert

### 🔹 Bài 2: Tính diện tích hình chữ nhật
- Nhập chiều dài, chiều rộng
- Tính và hiển thị diện tích

### 🔹 Bài 3: Tính chu vi và diện tích hình tròn
- Nhập bán kính
- Áp dụng công thức toán học để tính toán

### 🔹 Bài 4: Tính tiền điện
- Nhập chỉ số cũ, chỉ số mới, đơn giá
- Tính tiền điện theo công thức:
  > Tiền = (Chỉ số mới - Chỉ số cũ) × Đơn giá

### 🔹 Bài 5: Tính cạnh huyền tam giác vuông
- Nhập 2 cạnh góc vuông
- Áp dụng định lý Pitago:
  > c = √(a² + b²)
